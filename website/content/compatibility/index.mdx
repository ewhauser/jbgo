# Compatibility

gbash aims for high compatibility with existing shell ecosystems. These reports track conformance against upstream test suites and standards.

- [POSIX Shell](/docs/compatibility/posix) -- compliance matrix for the POSIX Shell Command Language (XCU Chapter 2)
- [Bash](/docs/compatibility/bash) -- conformance against the OILS spec test corpus, run against a pinned version of bash
- [Coreutils](/docs/compatibility/coreutils) -- coverage of the GNU coreutils test suite
